# Time Series Analysis Module

## Problem Statement
Time-based data appears in finance, weather forecasting,
sales tracking, and monitoring systems.

## Objectives
- Read and process time-series datasets
- Smooth data using moving averages
- Detect trends and seasonality
- Perform basic forecasting

## Features Implemented
- CSV-based time series reader
- Simple moving average smoothing
- Trend detection logic
- Seasonality pattern detection
- Naive forecasting model

## Outcome
This module provides foundational tools for time-series analysis
and prepares the system for advanced forecasting techniques.
