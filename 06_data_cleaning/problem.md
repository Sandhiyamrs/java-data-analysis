# Data Cleaning Module

## Problem Statement
Real-world datasets often contain missing values, duplicates, and outliers.
These issues reduce the accuracy of data analysis and models.

## Objectives
- Remove or replace missing values
- Detect and eliminate outliers
- Normalize data
- Remove duplicate entries

## Features Implemented
- Mean-based missing value handling
- Standard deviation based outlier detection
- Min-Max normalization
- Duplicate removal
- Central cleaning pipeline

## Outcome
This module prepares raw numerical data for accurate statistical analysis
and machine learning tasks.
